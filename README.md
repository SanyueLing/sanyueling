# 生日礼物

这是一个基于网页的互动解密游戏，玩家需要通过阅读剧情和解开谜题来推进游戏进程。

## 游戏特点

- **卷轴式界面**：采用传统的卷轴设计，黑色背景配米黄色卷轴
- **剧情驱动**：通过5个页面的剧情推进，讲述白依玲寻找哥哥的故事
- **解谜元素**：包含输入框谜题，需要玩家输入正确答案才能继续
- **进度保存**：自动保存游戏进度，下次打开可继续游戏
- **安全保护**：禁用右键和F12，防止作弊

## 游戏剧情

玩家扮演白依玲，在19岁生日当天收到一个神秘的木箱子，通过解谜寻找失联哥哥的线索...

## 技术实现

- **HTML5 + CSS3 + JavaScript**：纯前端实现
- **响应式设计**：支持桌面和移动设备
- **本地存储**：使用localStorage保存游戏进度
- **渐出动画**：文字和图片的优雅显示效果

## 如何运行

### 在线访问（推荐）
游戏已部署到GitHub Pages，可以直接在线访问：
**[https://你的用户名.github.io/sanyueling](https://你的用户名.github.io/sanyueling)**

### 本地运行
1. 启动本地服务器：
   ```bash
   python3 -m http.server 8000
   ```

2. 在浏览器中访问：`http://localhost:8000`

## 游戏玩法

1. 使用鼠标滚轮或方向键上下滚动浏览剧情
2. 遇到谜题时，在输入框中输入答案
3. 谜题答案：**木箱翻面**（第4页）
4. 解开谜题后可以继续向下滚动
5. 游戏进度会自动保存

## 设计文件格式

每个页面的设计文件使用类似XML的格式：

```xml
<txt>{txt1}</txt>
<pic>{pic1}</pic>
<inputbox>{secert1}</inputbox>
{
    txt1 = "./src/txt/txt1"
    pic1 = "./src/pic/pic1.png"
    secert1 = {
        "谜底" = "木箱翻面"
        "错误提示文案" = "一共是四个字"
    }
}
```

- `<txt>`：文本内容
- `<pic>`：图片内容  
- `<inputbox>`：谜题输入框
